--------- Help ----------------
CC : current control (in red)
MW : mouse wheel
CP : cursor position
-------------------------------
Space	: toggle pause
D	: toggle debug mode
R	: resets simulation
MW	: Zoom
Esc	: close window
C	: reload configuration
Arrows	: move simulation
	view
Tab	: switch to next
	control
PageUp	: increase CC
PageDown  : decrease CC
M         : load map
W         : add Worker at CP
A         : add Soldier at CP
T         : add Termite at CP
H         : add Anthill at CP
U         : increase temperature
L         : decrease temperature
